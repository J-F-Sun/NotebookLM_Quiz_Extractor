# Privacy Policy

Last updated: 2026-04-24

This Privacy Policy describes how **NotebookLM Quiz Extractor** handles user data.

## 1. What the extension does

NotebookLM Quiz Extractor is a Chrome extension that detects quiz content in NotebookLM quiz views and exports the current quiz to local JSON and Markdown files.

## 2. Data the extension accesses

The extension may access the following data only when the user opens NotebookLM and uses the extension:

- Quiz titles
- Quiz questions
- Multiple-choice options
- Correct answers
- Related quiz page content required for export

This data is treated as website content and is accessed only to perform the export requested by the user.

## 3. How the data is used

The extension uses the accessed content only to:

- detect the current NotebookLM quiz
- traverse quiz questions
- generate local export files in JSON and Markdown format

## 4. Data sharing

The extension does **not** sell, transfer, or share user data with third parties.

The extension does **not** upload quiz content to any external server controlled by the developer.

## 5. Local processing

All quiz detection, parsing, and export processing happens locally in the user's browser.

If the user chooses an export directory, the generated files are saved locally on the user's device.

## 6. Data retention

The extension does not maintain a remote database of user content.

The extension may store limited local preferences in the browser, such as:

- interface language
- last panel position
- selected export directory handle or display name, where supported by the browser

These settings are stored locally to improve usability and are not transmitted to the developer.

## 7. Permissions

The extension requests:

- `downloads`: used to save exported quiz files to the user's device
- host permissions for NotebookLM and related content frames: used only to detect and export quiz content rendered in NotebookLM and its related frames

## 8. Remote code

The extension does not use remote code. Its executable logic is packaged with the extension itself.

## 9. Children's privacy

The extension is not designed for children and does not intentionally collect personal information from children.

## 10. Changes to this policy

This Privacy Policy may be updated from time to time. Material updates will be reflected by updating the date at the top of this page.

## 11. Contact

If you have questions about this Privacy Policy, contact:

- GitHub repository issue tracker, or
- the contact email/address provided on the extension listing

# 隐私政策

最后更新：2026-04-24

本隐私政策说明 **NotebookLM Quiz Extractor** 如何处理用户数据。

## 1. 插件用途

NotebookLM Quiz Extractor 是一个 Chrome 扩展，用于识别 NotebookLM 测验视图中的题目内容，并将当前测验导出为本地 JSON 和 Markdown 文件。

## 2. 插件会访问哪些数据

仅当用户打开 NotebookLM 并主动使用本扩展时，扩展才可能访问以下数据：

- 测验标题
- 题目内容
- 选择题选项
- 正确答案
- 完成导出所必需的相关测验页面内容

这些数据属于网站内容，扩展仅为完成用户请求的导出功能而访问这些内容。

## 3. 数据用途

扩展访问这些内容仅用于：

- 检测当前 NotebookLM 测验
- 遍历整套题目
- 生成本地 JSON 和 Markdown 导出文件

## 4. 数据共享

本扩展**不会**向第三方出售、传输或共享用户数据。

本扩展**不会**将测验内容上传到开发者控制的外部服务器。

## 5. 本地处理

所有测验检测、解析和导出处理都在用户本地浏览器中完成。

如果用户选择了导出目录，生成的文件将保存在用户自己的设备上。

## 6. 数据保留

本扩展不会在远程服务器上保存用户内容。

本扩展可能会在浏览器本地保存少量偏好设置，以提升使用体验，例如：

- 界面语言
- 面板上次位置
- 已选择的导出目录句柄或目录显示名称（在浏览器支持的情况下）

这些设置仅保存在本地，不会传输给开发者。

## 7. 权限说明

本扩展请求以下权限：

- `downloads`：用于将导出的测验文件保存到用户设备
- NotebookLM 及相关内容 frame 的主机权限：仅用于识别并导出 NotebookLM 及相关 frame 中渲染的测验内容

## 8. 远程代码

本扩展不使用远程代码。扩展的可执行逻辑均随扩展安装包一并发布。

## 9. 儿童隐私

本扩展并非面向儿童设计，也不会有意收集儿童个人信息。

## 10. 政策更新

本隐私政策可能会不时更新。如有重要变更，将通过更新本页面顶部日期的方式反映。

## 11. 联系方式

如对本隐私政策有疑问，请通过以下方式联系：

- GitHub 仓库 issue 页面，或
- 插件商店页面中提供的联系邮箱/联系方式
